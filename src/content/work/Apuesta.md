##### Description:
This is a very basic project at first glance, developed solely with JavaScript. It consists of an interactive betting game where the user starts with a balance of 50 euros and must choose a number from 1 to 6 and the amount of money they want to bet. At the end, the user receives statistics of the plays and money lost once the balance reaches 0 and the game is lost.

#### Key Features:

##### User Interactivity: 
The game allows the user to actively participate by making decisions about the number they want to bet on and the amount of money to bet on each play.

##### Balance Tracking: 
The user's balance is tracked after each play, displaying the remaining balance after each bet and notifying the user when the balance reaches 0, indicating the end of the game.

##### Final Statistics: 
Once the user loses the game, detailed statistics are generated, including the number of plays made and the total money lost during the game.

#### Project Objectives:

Provide a simple yet engaging gaming experience for users, allowing them to actively participate and make strategic decisions.

Offer a fun way to learn and practice basic programming concepts in JavaScript, such as variable handling, control structures, and functions.

Help users understand the concept of risk management and probabilities through the practical experience of betting virtual money in an interactive game.
