##### Description:
This project is a website designed for searching Disney characters using a third-party API, developed entirely with JavaScript. Users can browse through an extensive database of Disney characters and access detailed information about their favorite characters.

#### Key Features:
##### Character Search: 
Users can search for Disney characters by entering keywords such as character name, movie title, or franchise into a search bar.

##### API Integration: 
The website utilizes a third-party API that provides a comprehensive database of Disney characters, including information about their appearances, backgrounds, and more.

##### Detailed Character Profiles: 
Each character listed in the search results is accompanied by a detailed profile, showcasing their name, image, description, and relevant information such as movies or TV shows they appear in.

##### Interactive User Interface: 
The user interface is designed to be intuitive and interactive, allowing users to easily navigate through search results and explore additional information about their favorite Disney characters.

#### Technologies Used:
JavaScript: Used for implementing the functionality of the website, including handling user interactions, fetching data from the API, and dynamically updating the content of the page.

#### Project Objectives:
Provide Disney fans with a convenient platform for exploring and learning about their favorite characters from various Disney movies and franchises.

Showcase proficiency in working with third-party APIs and integrating external data sources into web applications.

Create a user-friendly and visually appealing website that captures the magic and charm of the Disney universe.
